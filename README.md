# 🎯 Number Guessing Game

A fun and interactive number guessing game built with Python. Perfect for B.Tech first-year project!

## 🎮 Features

- **4 Difficulty Levels**: Easy, Medium, Hard, Extreme
- **Smart Hints**: Get hints after wrong guesses
- **Score System**: Earn points based on performance
- **Statistics**: Track your games and scores
- **Leaderboard**: Compete for high scores
- **Data Persistence**: Scores saved automatically

## 🎲 How to Play

1. Choose difficulty level
2. Guess the secret number
3. Get hints if stuck
4. Try to guess in fewest attempts
5. Earn points and beat high scores!

## 🚀 How to Run

```bash
python guessing_game.py
